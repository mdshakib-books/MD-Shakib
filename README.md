# SHAKIB Application

A production-ready full-stack application featuring a Vite + React frontend and Node + Express + MongoDB backend with Socket.io configuration.

## Features

- Scalable structured folders based on standard MVC formats
- Authentication boilerplate
- Realtime components via Socket.io
- Docker ready configurations

## Tech Stack

- **Frontend**: Vite, React, Socket.io-client
- **Backend**: Node.js, Express, MongoDB, Socket.io
- **Deployment**: Docker

## Getting Started

1.  **Clone the repository**
2.  **Start with Docker**: `docker-compose up --build`
3.  **Local setup**:
    - `cd server && npm install && npm start`
    - `cd client && npm install && npm run dev`
